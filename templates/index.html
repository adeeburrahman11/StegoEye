How Does Detection Work?

1. EOF Appendage Checks

Many amateur steganography tools simply append files or bytes to the literal end-of-file marker of JPEGs (FFD9) or PNGs (IEND). StegoEye scans binary headers to immediately flag trail data and provides full byte extraction downloads.

2. Wilson-Hilferty Chi-Square Test

Natural colors vary gradually, but LSB randomizations equalize neighboring pixel values. We compute Chi-Square values across 128 Color Pairs (PoVs) and map them using Wilson-Hilferty normal distribution formulas to generate a precise steganography threat percentage.

3. Interactive Bit-Slicing

By slicing the image into single-bit dimensions, we can isolate and render hidden patterns in the lower bit planes (LSBs) that are invisible to human sight. Random steganography stands out as sharp, uniformly dense static noise against structural objects.
